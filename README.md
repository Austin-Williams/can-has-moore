## Getting Started

```
$ sudo apt-get update
$ sudo apt-get install python-igraph python-numpy git -y
$ git clone https://github.com/Austin-Williams/can-has-moore.git
$ cd can-has-moore
$ tmux
$ python -i findMoore.py -a -s -n 57

```
